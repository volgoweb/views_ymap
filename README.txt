Отображает объекты (ноды) на яндекс карте.
При клике по ссылке в списке нод на карте показывается баллун выбранной ноды.

Для использования модуля:
1) В типе материала, который должен отображаться на карте, добавьте поля field_lat и field_lon для хранения координат.

2) Создайте вьюс для отображения карты.

3) Во вьюсе добавьте какому-нибудь полю (например, заголовоку ноды) класс  map-object-item_js и атрибуты lat, lon и nid. 
Вот пример вывода заголовка ноды:
<a href="#" class="map-object-item_js" lat="55.7113310000" lon="37.7610830000" nid="20458">ООО "САВА-сервис"</a>

4) Измените имя шаблона вьюса в каталоге tpl/ на подходящее под ваше имя вьюса. Или можете не использовать этот шаблон вовсе.

5) Добавьте на страницу в любое место <div id="services-ymap">

6) В первых строках файла модуля измените значения констант 
VY_OBJECT_NODE_TYPE_NAME (машинное имя типа материала, который должен отображаться на карте)
VY_MAP_VIEW_NAME (машинное имя вьюса для карты)
VY_MAP_DISPLAY_NAME (машинное имя дисплея вьюса для карты)
