Отображает объекты (ноды) на яндекс карте.
Ноды имеют поля field_lat и field_lon для хранения координат.
Вьюс отображает список данных нод и ссылку, которая содержит в аттрибутах координаты.
В подвале вьюса выводится карта.
При клике по ссылке в списке нод на карте показывается баллун выбранной ноды.
